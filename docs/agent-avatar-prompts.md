# エージェントアバター画像生成プロンプト（nanobanana Pro用）

各エージェントの円形アバター画像を生成するためのプロンプト集です。

## 共通設定
- **スタイル**: Pixar/Disney風の3Dアニメーション、子供向け、明るく親しみやすい
- **構図**: 正面向き、胸から上のポートレート、円形フレームに最適化
- **背景**: シンプルなグラデーション背景（各キャラクターのテーマカラー）
- **表情**: 笑顔、優しい目、子供に話しかけるような温かい表情
- **解像度**: 1024x1024px（円形クロップ前提）

## 生成時の注意点

### nanobanana Pro設定推奨値
- **Model**: Flux Pro 1.1 または Flux Dev
- **Aspect Ratio**: 1:1 (Square)
- **Steps**: 30-50
- **Guidance Scale**: 7-8
- **Seed**: ランダム（気に入った結果が出たらシード値を保存）

### 後処理
1. 生成された画像を1024x1024pxで保存
2. 円形クロップツールで中心を切り抜き
3. 背景を透過PNG化（オプション）
4. `/public/avatars/` フォルダに配置

### バリエーション生成のコツ
- 表情を変える: "warm smile" → "excited smile" / "gentle smile"
- 角度を変える: "facing forward" → "slight angle" / "three-quarter view"
- アクセサリーを追加: 各専門分野に関連する小物を持たせる
- 背景エフェクトを調整: 浮遊する要素の種類や数を変える

### 一貫性を保つために
- すべてのキャラクターで "Pixar 3D animation style" を維持
- "child-friendly" と "warm lighting" を必ず含める
- "High quality, detailed, professional 3D render" で品質を統一
- 各キャラクターのテーマカラーを背景に反映
